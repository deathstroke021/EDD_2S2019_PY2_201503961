#### Descripción

Proyecto 2 Estructura de datos 
Se desea que desarrolle la aplicación para el manejo de archivos para la Universidad de San Carlos de Guatemala de la facultad de Ingeniería. El sistema denominado como EDD DRIVE debe de llevar el control de usuarios, donde cada uno de los cursos de la carrera de ingeniería en sistemas debe de contar con un espacio de almacenamiento donde se puede subir, crear y eliminar carpetas, así como también archivos. Los usuarios también tendrán la opción de modificar los nombres de carpetas y archivos ya creados.
